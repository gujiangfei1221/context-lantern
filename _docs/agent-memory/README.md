# Agent Memory

This directory stores curated project memory for future AI sessions. It is not a raw chat archive.

## Files

- `thread-index.md`: source threads, local files, and known gaps used for memory curation.
- `decisions-and-changes.md`: durable workflow and architecture decisions.
- `operations-runbook.md`: repeatable procedures for handoff and memory curation.
- `historical-issues.md`: retrospectives for repeated or important issues.

## Use Rules

- Read this directory when working on historical issues, task handoffs, AI workflow design, or memory curation.
- Keep `AGENTS.md` short; put details here.
- Put temporary current-task state in `_docs/handoff/`.
- Omit sensitive values from all memory files.
